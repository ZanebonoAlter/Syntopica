# 订阅源与分类管理

## 分类

- 新增分类
- 编辑分类
- 删除分类
- 分类可配置名称、图标、颜色、描述

> 删除分类时，不会删除该分类下的订阅源。

## 订阅源（Feed）

- 新增订阅源
- 编辑订阅源
- 删除订阅源
- 单个订阅源手动刷新
- 全量刷新
- OPML 导入
- OPML 导出

每个订阅源还带以下能力开关：

- `ai_summary_enabled` — 是否参与 AI 聚合摘要
- `article_summary_enabled` — 是否开启文章级 AI 内容整理
- `completion_on_refresh` — 刷新后是否立即处理内容补全
- `max_completion_retries` — 内容补全最大重试次数
- `firecrawl_enabled` — 是否开启全文抓取
