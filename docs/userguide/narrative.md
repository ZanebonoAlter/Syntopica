# 叙事面板

叙事面板是 Topic Graph 页面中的一项功能，帮助用户按时间线理解事件的发展脉络。

## 核心概念

- **SemanticBoard（语义板块）**：长期存在的主题分类，将相关事件组织在一起
- **NarrativeBoard（叙事板）**：每日生成的叙事实例，基于当日文章自动生成
- **Board 时间线**：按日期查看各语义板块的叙事摘要

## 功能

- **叙事面板**：查看每日叙事摘要和 Board 时间线
- **分类版块切换**：按分类查看 Board 列表，支持全局和分类两种视图模式
  - 全局模式：查看所有语义板块
  - 分类模式：按分类筛选，展示各分类下的板块数量
- **板块概念管理**：支持手动创建和 LLM 建议两种概念创建方式
  - 手动创建：自定义板块概念
  - LLM 建议：由 AI 分析后推荐新的板块概念
  - 编辑、停用已有概念
- **Board 可视化画布**：基于语义板块关联关系，可视化展示叙事板的组成和关联
- **叙事重新生成**：支持对指定日期重新生成叙事摘要

## 使用场景

1. 打开 Topic Graph 页面
2. 切换到叙事面板查看每日叙事摘要
3. 按分类切换查看不同领域的叙事
4. 浏览 Board 时间线了解事件发展脉络
5. 管理板块概念，优化叙事组织
