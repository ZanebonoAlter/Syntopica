# 文章标签

## 标签展示

文章正文区会展示由 AI 提取的 topic tags。这些标签帮助用户快速了解文章涉及的主题领域。

标签支持以下能力：

- **标签合并**：相似标签可以合并，减少冗余
- **抽象层级**：标签按层级组织，从具体到抽象
- **标签质量评分**：标签有质量评分，可过滤低质量标签
- **关注标签**：可以关注特定标签，关注后相关内容会在专题中优先展示

## 标签入口

- 文章正文区直接展示当前文章的标签
- Topic Graph 页面提供完整的标签浏览和管理界面
- digest 列表和详情中也展示聚合标签

## 语义板块文章匹配详情

在语义板块管理页选择某个板块后，切到「文章」Tab：

- 每篇文章的 tag chip 会显示匹配方式颜色和分数。
- 点击某个 tag chip，右侧会打开「匹配详情」面板。
- 面板展示当前存储的匹配方式/分数、公式解释、辅助标签逐对相似度，以及当前匹配参数。
- 点击同一个 tag chip 或面板关闭按钮可收起详情。
